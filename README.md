# erpel

Free and libre alternative to [Rambox](https://rambox.app/) and [franz](https://meetfranz.com/).

## Development

Use [just](https://github.com/casey/just) to execute commands. Run `just` to list all available commands.

### Installation

```shell
just install
```

### Run

```shell
just run
```

### Build

```shell
just build
```
